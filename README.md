# Blinky V1
A led chaser circuit using NE555 and CD4017 ICs which have a configurable speed using a 100K Ohm potentiometer

# Blinky - An LED chaser circuit using ICs only.
**The ICs used are:**
| Serial No.               | Part Name              | Purpose                 |
|:-------------------------|:----------------------:|:-----------------------:|
|1                         |NE555                  | Its the timer ic used to change the chase speeed.     |
|2                         |CD4017                  | It acts as a controller for controlling the LEDs.     |

Its a simple project made without any microcontrollers and just complete crude electronics 
It was a fun project to make learned quite a bit building the circuit.
